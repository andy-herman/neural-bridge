## How to query the wiki (for agents)

1. Read this index first.
2. Check `concepts/` for any pre-compiled articles on the topic.
3. Check the relevant `agents/<role>/` for raw session notes.
4. If nothing exists, search `daily-logs/` for relevant session summaries.
5. Synthesize. File the answer back into the wiki (your agent's subdirectory) when appropriate.
